# 🌀 重力反转跑酷 (Gravity Flip Runner)

一个创意十足、可玩性极高的休闲网页游戏！

## 🎮 游戏特色

- **创新玩法**：点击屏幕或按空格键反转重力方向，在天花板和地板之间自由切换
- **横冲直撞**：连按 Shift（手机连点 ⚡ 按钮）无视碰撞直线穿越，撞碎障碍物
- **连击系统**：连续吃金币触发连击，得分最高翻 5 倍
- **道具系统**：🛡️护盾(带倒计时)、🧲磁铁、⏳慢动作、⚡充能
- **动态难度**：速度快速攀升，重力也随之变强
- **打击音效**：Web Audio 实时合成音效，撞击/连击/横冲各有反馈
- **在线排行榜**：输入姓名登录，每局结束自动上传成绩并刷新全局排名
- **手机适配**：触屏完整可玩，专属横冲按钮，刘海屏安全区适配

## 🚀 快速开始

### 在线模式（含排行榜，需 Node.js ≥ 22.5）

```bash
node server.js
# 浏览器打开 http://localhost:3000
```

服务零依赖（用 Node 内置 `http` + `node:sqlite`），无需 `npm install`。
排行榜数据保存在自动生成的 `leaderboard.db`（SQLite）中，所有玩家共享。

### 纯前端模式（无排行榜）

直接双击 `index.html` 也能玩，但排行榜功能需要后端服务才生效。

## 🎯 游戏规则

1. **操作方式**：
   - 电脑：空格/点击 反转重力，连按 `Shift`/右键 横冲，`M` 静音
   - 手机：点击屏幕 反转重力，连点右下角 ⚡ 按钮 横冲

2. **得分方式**：
   - 躲过障碍物：+10 分
   - 收集金币：+25 分 ×连击倍率(最高 x5)
   - 横冲撞碎障碍物：+5 分
   - 存活时间也会增加分数

3. **游戏目标**：
   - 尽可能长时间存活，冲上排行榜榜首

## 📁 文件结构

```
休闲游戏/
├── index.html        # 游戏主页面(登录/排行榜/游戏)
├── style.css         # 样式文件(含移动端适配)
├── game.js           # 游戏逻辑 + 排行榜/触屏交互
├── server.js         # 后端服务(静态托管 + 排行榜 API)
├── package.json      # 项目元信息
├── leaderboard.db    # 运行时自动生成的 SQLite 数据库
└── README.md         # 说明文档
```

## 🛠️ 技术栈

- 前端：原生 HTML5 + CSS3 + JavaScript (Canvas 2D + Web Audio)
- 后端：Node.js 内置 `http` + `node:sqlite`（零第三方依赖）
- 数据库：SQLite

## 🌐 部署到云服务器

```bash
# 上传整个文件夹到服务器后
node server.js
# 或用 pm2 守护进程
pm2 start server.js --name gravity-game
# 自定义端口
PORT=80 node server.js
```

排行榜 API：
- `GET  /api/leaderboard?limit=10` —— 获取榜单
- `POST /api/score {name,score,coins}` —— 提交成绩，返回排名

## 🔧 自定义配置

可以在 `game.js` 文件的 `CONFIG` 对象中调整游戏参数：

```javascript
const CONFIG = {
    playerSize: 30,           // 玩家大小
    gravity: 0.6,            // 重力强度
    jumpForce: 12,           // 跳跃力度
    obstacleGap: 250,        // 障碍物间距
    initialSpeed: 3,         // 初始速度
    speedIncrease: 0.0005,   // 速度增长率
    maxSpeed: 8              // 最大速度
};
```

## 💡 游戏技巧

1. **节奏感**：掌握反转重力的节奏，不要过早或过晚
2. **预判**：提前观察即将到来的障碍物位置
3. **金币权衡**：评估收集金币的风险，不要为了金币而撞上障碍物
4. **速度适应**：随着游戏速度加快，反应需要更快
